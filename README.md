# 🏭 工业质检与 ERP 数据自动化协同 Agent

基于大语言模型（LLM）构建的“工业质检与 ERP 数据自动化协同 Agent”，主要用于解决传统产线质检（FQC）及国际贸易订单流转中的数据壁垒问题。

## 🎯 核心痛点

在传统的制造与贸易流程中，处理以下环节时高度依赖人工：
1. **复杂的电子产品规格书（BOM）解析**：非标准化文档难以自动化提取。
2. **海量质检日志的异常分类**：人工核对显微镜质检记录耗时长、易出错。
3. **ERP 系统的繁琐数据录入**：缺乏标准化的逻辑判断，导致数据流转出现断层。

## 🧠 核心逻辑流

系统采用了 **Agent 架构**并依赖**长链推理**：

1. **文档解析 Agent**：负责读取非标准化的产品规格书和冗杂的显微镜质检记录。
2. **逻辑推理 Agent**：进行长上下文的推理与核对，将非结构化数据精准映射为标准 BOM 格式，并根据多维度的质量参数判断合格率。
3. **ERP 自动化协同**：自动生成符合 ERP 系统 API 标准的 JSON 报文，实现数据自动入库或异常打回。
